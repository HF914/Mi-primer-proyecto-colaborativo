# Mi primer proyecto colaborativo

## Objetivo del proyecto
Aprender a utilizar herramientas de control de versiones y plataformas colaborativas para simular el entorno de trabajo en desarrollo de software.

## Herramientas colaborativas investigadas
- **GitHub**: Plataforma de alojamiento para control de versiones y colaboración.
- **GitKraken**: Interfaz gráfica para gestionar Git de forma visual.
- **Trello**: Para la comunicación y gestión de tareas en equipo.
